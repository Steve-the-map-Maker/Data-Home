Census AI Data Visualizer

Interact with the live application here: https://census-ai-frontend.onrender.com/

The Census AI Data Visualizer is a sophisticated application designed to provide an AI-driven interface for querying and visualizing data from the U.S. Census Bureau API. Unlike traditional dashboards, this application allows users to interact with census data using natural language, making complex statistical information accessible to a broader audience.

Key Features

  • AI-Powered Natural Language Processing: Allows users to query U.S. Census data using natural language, integrating initial LLM capabilities with function calling. The backend is resilient, scalable, and geographically-aware, leveraging AI-driven tools to understand user queries.
  • Comprehensive Census Data Access: Provides access to over 17 variables across demographics, economics, housing, and education from the U.S. Census Bureau ACS 5-Year Estimates.
  • Geographic Intelligence: Features a flexible engine that understands a wide range of Census geographies, their relationships, and user-friendly aliases. It handles hierarchical geography queries, such as retrieving county data within a specified state.
  • Two-Step AI Orchestration: Utilizes an advanced two-step process where the AI (Google Gemini Pro) generates tool calls, executes them to retrieve data, and then interprets the raw results to provide high-quality, natural language answers to the user.
  • Client-Side Data Visualization: Operates as a frontend-centric application that loads local GeoJSON files. All data merging and interactive map rendering are performed in the browser, empowering the frontend to be highly responsive. It uses a reusable `MapDisplay.jsx` component that dynamically renders choropleth maps for any valid GeoJSON with data.
  • Data Enrichment and Derived Metrics: The backend performs server-side calculations for complex, comparative questions (e.g., "Which states have more men than women?"). It supports derived metrics such as Male-Female Population Difference, Unemployment Rate (%), and Owner-Occupied Housing Rate (%).
  • Enhanced User Experience: Includes visual loading indicators during query processing, displays clear and user-friendly error messages, and provides initial guidance and example queries to users. Maps feature dynamic legends and interactive tooltips/popups for detailed data display. The design is responsive across different screen sizes.

Technology Stack

This application is built with a robust and modern technology stack:

  • Backend:
    • Python 3.12+: The core logic is implemented in Python.
    • FastAPI: A modern, high-performance web framework for building APIs.
    • Uvicorn: An ASGI server for FastAPI, enabling asynchronous operations.
    • Google Gemini Pro: The Large Language Model (LLM) used for natural language understanding and response generation.
    • U.S. Census Bureau API: The primary data source for demographic and economic data.
    • httpx: An asynchronous HTTP client for making API calls.
    • python-dotenv: Used for secure environment variable management.
    • Docker: For containerizing the backend application, ensuring portability and reliable deployment.
  • Frontend:
    • React with Vite: A JavaScript library for building user interfaces, with Vite as a fast development build tool.
    • Leaflet: An open-source JavaScript library for mobile-friendly interactive maps.
    • React-Leaflet: Provides React components for Leaflet maps.
    • chroma-js: Used for dynamic color scale generation for map visualizations.
    • axios: A promise-based HTTP client for making API requests to the backend.

How It Was Made

The Census AI Data Visualizer is a full-stack application composed of two distinct services: a backend API and a frontend web interface.

  • Backend Development: The Python backend handles all the heavy lifting, including natural language processing, API interactions with the Census Bureau, and data interpretation. It was designed to be a scalable and efficient service.
  • Frontend Development: The React.js frontend provides an intuitive user interface for submitting queries and displaying the results, communicating with the deployed backend API.
  • Deployment: Both the backend and frontend components are deployed on Render, a cloud application platform, ensuring the application is publicly accessible and scalable. The `render.yaml` file defines the deployment configuration for the backend service.

This project showcases the integration of advanced AI capabilities with governmental datasets to create a powerful and user-friendly tool for data exploration and understanding.
