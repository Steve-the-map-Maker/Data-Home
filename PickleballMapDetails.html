Pickleball Courts Map: Data Overview

Welcome to an interactive web application designed to help users discover pickleball court locations in the Portland area. This map provides a dynamic visualization of court data and offers various filtering options to enhance your search.

Key Features

  • Interactive Map Display: Visualizes pickleball courts on a dynamic MapLibre map.
  • Geolocation: Allows users to find courts near their current location.
  • Advanced Filtering:
    • Filter courts by distance from the user (1km to 100km).
    • Filter by court type (Indoor, Outdoor, or Any).
    • Filter by the minimum number of courts available at a location (1 to 8+).
  • Dynamic Sidebar:
    • Lists courts based on the current map view and active filters.
    • Displays detailed information for a selected court in a slide-in panel, including name, location, and number of courts.
  • "Get Directions": Provides a link to Google Maps for directions to the selected court, pre-filling the user's current location if available.
  • Dynamic Map Marker Styling: Map markers are styled based on:
    • **Size:** Reflects the number of courts at the location (larger circles for more courts).
    • **Color:** Indicates court type (Blue for Indoor, Green for Outdoor, Purple for Both).
  • Custom UI Controls: Includes a custom-styled dropdown for selecting court type with color swatches, and a visual filter for minimum courts using clickable circles.
  • Responsive Design: Adapts to different screen sizes, including mobile devices.

How the Map Was Made

The interactive map of pickleball courts was created using several tools and data sources to provide an accurate and user-friendly experience.

  • Mapping Library: The map is built using MapLibre GL JS, an open-source mapping library that allows for the creation of high-quality, interactive maps.
  • Basemap: The basemap used is from OpenStreetMap, a community-driven and detailed geographic data provider.
  • Frontend: The application is developed with HTML5, CSS3, and JavaScript (ES6+).
  • Data Format: The court location data is sourced from a GeoJSON file (`PortlandPickelBallData.geojson`), which is loaded dynamically by the application.

The process involved integrating these data sources and technologies, performing data cleaning and transformation, and ultimately visualizing the data in an interactive manner that makes it easy to find pickleball courts across the Portland area.
