Fantasy Football App

Interact with the live application: View Live App

This React.js application is designed to help fantasy football enthusiasts analyze player data and make informed decisions for their teams. It provides various components for viewing player information, comparing players, and charting performance trends.

Key Features:

  • Player Data Visualization: Displays fantasy-relevant data, weekly offense statistics, and yearly offense statistics for various players.
  • Interactive Components: Features interactive tables, player cards, and charts to visualize player performance and rankings.
  • Player Comparison Tool: Allows users to compare different players side-by-side based on key metrics.
  • Built with React: The application is developed using React.js for a dynamic and responsive user experience.
  • Chart.js Integration: Utilizes Chart.js and react-chartjs-2 for creating compelling data visualizations.

The project demonstrates skills in frontend development, data integration from CSV and JSON sources, and creating a user-friendly interface for data analysis.
