Fantasy Football App

Live App: Steve-the-map-Maker.github.io/fantasy-football-app

Project Overview

This React.js application is designed to help fantasy football enthusiasts analyze player data and make informed decisions for their teams, enabling data-driven draft decisions and weekly matchup optimizations. It provides various components for viewing player information, comparing players, and charting performance trends.

Key Features

  • Player Data Visualization: Displays fantasy-relevant data, weekly offense statistics, and yearly offense statistics for various players.
  • Interactive Components: Features interactive tables, player cards, and charts to visualize player performance and rankings.
  • Player Comparison Tool: Allows users to compare different players side-by-side based on key metrics.

Technology Stack

  • React.js
  • Chart.js & react-chartjs-2
  • Data sourced from CSV and JSON files
