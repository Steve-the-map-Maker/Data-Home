Los Angeles and Orange County Income Distribution: Data Overview

Welcome to my interactive analysis of income levels across Los Angeles and Orange County. This dashboard helps visualize the median income of each census tract and provides an understanding of the economic landscape in the region.

Income Levels Across Census Tracts

My data breaks down the median income for different census tracts across LA and Orange County:

  • Below $30,000: Areas with the darkest shades of green indicate higher income levels.
  • $30,000 - $60,000: These areas are represented by a medium shade of green, showing moderate income levels.
  • $60,000 - $90,000: Lighter shades of green represent census tracts with median incomes falling in this range.
  • Above $90,000: The lightest areas represent the regions with the lowest income levels.

The map offers a dynamic way to interact with the data, showing trends and disparities across the regions of interest.

Visual Analysis

Interactive Map: The interactive web map provides a detailed spatial breakdown of the median income distribution across the Los Angeles area. The gradient colors visually represent varying income levels, with darker greens indicating higher incomes and lighter shades indicating lower ones.

Income Distribution Chart: The chart displays income distribution within the region, highlighting the number of census tracts in each income bracket.

These visualizations aim to make it easier to understand income inequalities and the diversity of economic statuses across different neighborhoods in Los Angeles and Orange County.

How the Map Was Made

The interactive map was created using several tools and data sources to provide an accurate representation of income distribution across Los Angeles and Orange County.

  • Mapping Library: The map is built using MapLibre, an open-source mapping library that allows for high-quality, interactive visualizations.
  • Basemap: The basemap used is from OpenStreetMap, which provides a detailed and community-driven geographic base layer.
  • Frontend Framework: The application is developed using React, a powerful JavaScript library for building user interfaces, which enables dynamic rendering of map components.
  • Data Sources: The data used in the map comes from the U.S. Census Bureau. Specifically, it combines two different datasets: census tract data and income data. These datasets were joined using census tract identifiers to provide a complete view of the income distribution.

The process of creating the map involved integrating these data sources and technologies, performing data cleaning and transformation, and ultimately visualizing the data in an interactive manner that makes it easy to understand spatial patterns in income across the region.
